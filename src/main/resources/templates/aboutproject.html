About This Project

Project Requirements

  • Using the PITCH specification document, create an interface that allows a user to upload the attached PITCH data file.
  • The application should process the data and produce output showing a count of messages by message type.
  • Each line in the attached file begins with an extra 'S' character that is not mentioned in the spec. This character can be ignored.
  • "Feel free to use the language and framework of your choice."
  • Return the completed code with instructions for execution.
